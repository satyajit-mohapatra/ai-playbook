# Create Learning Roadmaps

Guidelines, best practices, and a structured framework for creating structured learning roadmaps for technical skills with phased milestones, resources, and career guidance.

## Roadmap Structure

### 1. Overview
- Topic introduction
- Career relevance
- Estimated time to proficiency
- Prerequisites

### 2. Foundation Phase
- Core concepts to learn
- Essential terminology
- Basic skills to master
- Suggested resources
- Milestone projects

### 3. Intermediate Phase
- Advanced concepts
- Practical application skills
- Common patterns and practices
- Tool proficiency
- Portfolio projects

### 4. Advanced Phase
- Expert-level concepts
- Architectural thinking
- Performance optimization
- System design skills
- Capstone projects

### 5. Specialization Paths
- Role-specific tracks
- Industry-specific applications
- Emerging trends
- Advanced topics

### 6. Resources & References
- Courses and tutorials
- Books and articles
- Practice platforms
- Communities and forums
- Certification options

## When to Apply

Use this framework when:
- Planning skill development paths
- Creating training curricula
- Guiding career development
- Organizing self-learning plans

## Expected Output

When applying this framework, include:
1. Phased learning plan
2. Weekly or monthly milestones
3. Project recommendations
4. Resource links
5. Skill assessment checkpoints
6. Career path guidance
