# Write Professional Emails

Guidelines, best practices, and a structured framework for composing effective emails for various scenarios including technical, persuasive, and communication emails.

## Email Types

### 1. Professional Emails
- Clear subject line
- Professional greeting
- Concise body
- Call to action
- Professional closing

### 2. Technical Emails
- Technical context
- Issue description
- Steps to reproduce
- Expected vs actual
- Attachments and logs

### 3. Communication Emails
- Meeting requests
- Status updates
- Feedback delivery
- Announcement emails
- Apology/follow-up

### 4. Persuasive Emails
- Value proposition
- Benefit highlights
- Call to action
- Urgency creation
- Closing hook

## Best Practices

### Structure
- Compelling subject line
- Personalized greeting
- Clear purpose statement
- Concise, scannable content
- Strong call to action
- Professional signature

### Tone
- Professional but approachable
- Clear and direct
- Respectful of time
- Appropriate formality
- Positive language

### Content
- Lead with the main point
- Use short paragraphs
- Include specific details
- Provide context
- Set clear expectations

## When to Apply

Use this framework when:
- Writing professional correspondence to colleagues or clients
- Communicating technical issues or bug reports
- Sending meeting requests or status updates
- Delivering feedback or announcements
- Composing persuasive proposals or requests

## Expected Output

When applying this framework, include:
1. Complete email
2. Subject line options
3. Key points summary
4. Tone assessment
5. Alternative versions
6. Follow-up suggestions
