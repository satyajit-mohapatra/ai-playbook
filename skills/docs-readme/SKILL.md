# Write README Documentation

Guidelines, best practices, and a structured framework for generating comprehensive project README documentation including installation, usage, API reference, and contributing guidelines.

## README Structure

### 1. Project Title & Badges
- Clear, descriptive title
- Build status, version, license badges
- One-line description
- Key metrics (stars, downloads)

### 2. Description
- What the project does
- Problem it solves
- Key features (3-5 bullet points)
- What makes it unique
- Screenshots/media if applicable

### 3. Table of Contents
- Navigation for longer READMEs
- Links to all major sections

### 4. Installation
- Prerequisites
- Step-by-step installation
- Platform-specific notes
- Verification steps

### 5. Quick Start
- Minimal working example
- Common use cases
- Code examples with explanations
- Expected output

### 6. Usage
- In-depth usage examples
- Configuration options
- Environment variables
- CLI commands if applicable

### 7. API Documentation
- Key functions/methods
- Parameters and return values
- Usage examples
- Type definitions

### 8. Examples
- Real-world scenarios
- Links to example projects
- Tutorials

### 9. Configuration
- Config file examples
- Environment variables
- Options reference

### 10. Contributing
- Link to contributing guide
- Developer setup
- Running tests

### 11. Testing
- How to run tests
- Test coverage
- CI/CD information

### 12. Roadmap
- Planned features
- Known issues
- Future directions

### 13. License
- License type
- Copyright notice
- Link to full license

### 14. Support
- Issue tracker link
- Contact information
- Community channels

## When to Apply

Use this framework when:
- Creating new project documentation
- Updating existing README files
- Onboarding users to open source projects
- Documenting library or tool usage

## Expected Output

Provide complete, production-ready README with:
1. All sections above
2. Proper Markdown formatting
3. Working code examples
4. Visual elements where helpful
5. Links to related resources
